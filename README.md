# Left Ovr Ai Website

A simple, responsive website for showcasing the Left Ovr Ai application, providing information, and linking to downloads.

## Features

- Custom logo and floating header design
- Responsive design that works on desktop, tablet, and mobile devices
- Modern, clean UI with smooth animations
- Dedicated downloads page with system requirements
- Showcase area to highlight app features or screenshots
- Easy to customize and extend

## File Structure

- `index.html` - Main landing page
- `downloads.html` - Page for app downloads
- `styles.css` - Stylesheet for the entire site
- `script.js` - JavaScript for interactive elements

## How to Use

1. Replace placeholder text and images with information about your app
2. Update download links in the downloads.html file
3. Add real showcase images in the showcase section
4. Customize colors and styles in the CSS file as needed

## Customization

### Changing Colors

The primary color scheme can be modified in the `styles.css` file:
- Primary color: `#4a6bff`
- Secondary colors: `#6b8dff`, `#8da9ff`
- Background colors: `#f8f9fa`, `#fff`
- Text colors: `#333`, `#2c3e50`, `#6c757d`

### Adding Pages

To add more pages, create new HTML files based on the structure of the existing pages and update the navigation links in all HTML files.
